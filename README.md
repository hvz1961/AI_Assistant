# 🪟 ИИ-ассистент для компании по установке окон

## 📋 Описание проекта

**Название проекта:** ИИ-ассистент для компании по установке окон  
**Клиент:** Компания «Народные окна»  
**Дата создания:** 2026  
**Статус:** ✅ Реализован

---

## 🎯 Цель проекта

Разработка интеллектуального ИИ-ассистента для автоматизации первичной коммуникации с потенциальными клиентами компании по установке металлопластиковых окон. Главная цель — **конвертация диалога в запись на бесплатный замер** с последующей синхронизацией встречи в Google Календарь.

## 🎭 Сценарии диалогов

### Основной сценарий (Happy Path)

```
1. ПРИВЕТСТВИЕ
   └─→ "Здравствуйте! Я ваш помощник из «Народные окна»..."

2. ВЫЯВЛЕНИЕ ПОТРЕБНОСТИ
   └─→ "Что нужно остеклить — квартиру, дом или балкон?"

3. ПРЕЗЕНТАЦИЯ ВЫГОДЫ
   └─→ "У нас собственное производство, гарантия до 5 лет..."

4. ПРЕДЛОЖЕНИЕ ЗАМЕРА
   └─→ "Замер бесплатный, ни к чему не обязывает..."

5. СБОР КОНТАКТОВ
   └─→ Имя, телефон, адрес, удобное время

6. ПОДТВЕРЖДЕНИЕ И СИНХРОНИЗАЦИЯ
   └─→ Создание встречи в Google Calendar
```

### Сценарий отработки возражений

| Возражение | Скрипт ответа |
|------------|---------------|
| «Дорого» | Акцент на собственное производство, рассрочка до 24 месяцев |
| «Я подумаю» | Замер бесплатный — точные цифры для сравнения |
| «Пришлите расчёт по фото» | Фото не даёт точности, замерщик приедет бесплатно |
| «Уже заказал в другом месте» | Оставить контакты на будущее, отправить каталог |
| «Читал плохие отзывы» | Рейтинг 5.0 на Яндекс.Картах (149 оценок), 18+ лет на рынке |

## 📅 Интеграция с Google Calendar

### Параметры создаваемой встречи

| Поле | Значение |
|------|----------|
| **Название** | Замер окна — {Имя клиента} |
| **Описание** | Тип объекта: {квартира/дом/балкон}<br>Адрес: {адрес}<br>Телефон: {телефон}<br>Комментарий: {описание задачи} |
| **Длительность** | 30 минут |
| **Напоминание** | За 2 часа до встречи (SMS/Email) |
| **Статус** | Подтверждается менеджером |

### Пример JSON для Google Calendar API

```json
{
  "summary": "Замер окна — Иван Петров",
  "description": "Тип объекта: Квартира\nАдрес: Москва, ул. Ленина, д. 10\nТелефон: +7 999 123-45-67\nКомментарий: 2 окна, балкон",
  "start": {
    "dateTime": "2026-02-15T10:00:00+03:00",
    "timeZone": "Europe/Moscow"
  },
  "end": {
    "dateTime": "2026-02-15T10:30:00+03:00",
    "timeZone": "Europe/Moscow"
  },
  "attendees": [
    {
      "email": "manager@n-okna.ru",
      "responseStatus": "needsAction"
    }
  ],
  "reminders": {
    "useDefault": false,
    "overrides": [
      {
        "method": "email",
        "minutes": 120
      }
    ]
  }
}
```

## 🔐 Безопасность и конфиденциальность

| Аспект | Реализация |
|--------|------------|
